# Section 102.4

## Canonical Code Text

102.4 Application of building code. The design 
and construction of new structures shall comply 
with SBC 201 and any alterations, additions, 
changes in use or changes in structures required by 
this code, which are within the scope of SBC 201, 
shall be made in accordance therewith. 
❖ 
The code is the companion fire and life safety 
maintenance code to SBC 201. Maintenance of 
other building features is governed by other 
Saudi Codes. When existing buildings change 
occupancy group or are altered, increased in 
area or demolished, SBC 201 provisions must 
apply. When compliance with the code requires 
alterations, additions or modifications within 
the scope of SBC 201, SBC 201 regulations 
and the fire official’s authority must prevail. This 
makes it essential that the building and fire 
officials responsible for enforcing the building 
and fire codes establish a sound working 
relationship. Clear communication is essential 
to achieve compliance with the respective 
building and fire officials’ orders.
