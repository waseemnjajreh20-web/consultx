# Section 114.3

## Canonical Code Text

114.3 Record.  The Fire official shall cause a 
report to be filed on an unsafe condition. The report 
shall state the occupancy of the structure and the 
nature of the unsafe condition. 
❖ 
The Fire official must file a report on each 
investigation of unsafe conditions, stating the 
occupancy of the structure and the nature of 
the unsafe condition. This report provides the 
basis for the notice described in Section 114.4.
