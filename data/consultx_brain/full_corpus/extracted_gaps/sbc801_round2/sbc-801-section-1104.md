# Section 1104

## Canonical Code Text

SECTION 1104 MEANS OF EGRESS 
FOR EXISTING BUILDINGS 
1104.1 General. Means of egress in existing 
buildings shall comply with the minimum egress 
requirements where specified in Table 1103.1 as 
further enumerated in Sections 1104.2 through 
1104.25, and the building code that applied at the 
time of construction. Where the provisions of this 
chapter conflict with the building code that applied 
at the time of construction, the most restrictive 
provision shall apply. Existing buildings that were 
not required to comply with a building code at the 
time of construction shall comply with the 
minimum egress requirements where specified in 
Table 1103.1 as further enumerated in Sections 
1104.2 through 1104.25. 
❖ 
The primary concept of this section is to require 
existing buildings to comply with the specific 
means of egress requirements for new 
buildings except as modified by this section.
