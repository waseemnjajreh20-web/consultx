# Section 302.1

## Canonical Code Text

302.1 Definitions. The following terms are 
defined in Chapter 2: 
• 3D PRINTER. 
• ADDITIVE MANUFACTURING 
• BONFIRE. 
• HI-BOY. 
• HIGH-VOLTAGE TRANSMISSION LINE. 
• OPEN BURNING. 
• PORTABLE OUTDOOR FIREPLACE.  
• POWERED INDUSTRIAL TRUCK.  
• RECREATIONAL FIRE. 
• SKY LANTERN. 
❖ 
Definitions 
of 
terms 
can 
help 
in 
the 
understanding and application of the code 
requirements. This section directs the code 
user to Chapter 2 for the proper application of 
the indicated terms used in this chapter. Terms 
may be defined in Chapter 2 or in another SBC 
as indicated in Section 201.3, or the dictionary 
meaning may be all that is needed (see also 
commentaries, Section 201 through 201.4).
